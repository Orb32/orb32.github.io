Customizable Hardware

You’re orb32 A1 mini is customisable Here’s how

This device is built for makers/users. You can program it from scratch or drop in our starter code to move faster. Either way, you get full control, clean hardware access, and a smooth upgrade path as your ideas grow.

Two ways to get started

Choose the path that fits your workflow. You can build every line yourself or customize our base project and ship quicker.

Upload Code Use Existing Code

1. Upload code to it

You will write code from the ground up and compile it directly to the board.

Step 1

Download VSCode and install the PlatformIO IDE extension.

Step 2

Click New Project inside PlatformIO.

Step 3

Select Board: Espressif ESP32-C3-DevKitM-1 and choose Framework: Arduino. Then click Finish.

Step 4

Make sure you update your platformio.ini file exactly like this.

; PlatformIO Project Configuration File
;
;   Build options: build flags, source filter
;   Upload options: custom upload port, speed and extra flags
;   Library options: dependencies, extra library storages
;   Advanced options: extra scripting
;
; Please visit documentation for the other options and examples
; https://docs.platformio.org/page/projectconf.html

[env:esp32-c3-devkitm-1]
platform = espressif32
board = esp32-c3-devkitm-1
framework = arduino
lib_deps = 
	adafruit/Adafruit GFX Library
	adafruit/Adafruit ST7735 and ST7789 Library
	bblanchon/ArduinoJson@^7.4.2

Step 5

In your src/main.cpp file, start with this basic "hello world" code. Enjoy coding!

Show hello world code 
#include <Arduino.h>
#include <SPI.h>
#include <Adafruit_GFX.h>
#include <Adafruit_ST7789.h>
#include <ArduinoJson.h>

//For the ST7789 display, we need to define the pins for SCLK, MOSI, RST, DC, CS, and BLK (backlight).
#define TFT_SCLK 4
#define TFT_MOSI 6
#define TFT_RST 7
#define TFT_DC 8
#define TFT_CS 9
#define TFT_BLK 10

Adafruit_ST7789 tft(TFT_CS, TFT_DC, TFT_RST);

//For the encoder, we need to define the pins for ENC_A, ENC_B, and ENC_BTN (encoder button).
#define ENC_A 2
#define ENC_B 3
#define ENC_BTN 5
#define KO_BTN 1

void setup() {
  Serial.begin(115200);
  pinMode(TFT_BLK, OUTPUT);
  digitalWrite(TFT_BLK, HIGH);
  SPI.begin(TFT_SCLK, -1, TFT_MOSI, TFT_CS);
  tft.init(240, 320);
  tft.setRotation(3);
  tft.invertDisplay(false);
}

void loop() {
  tft.fillScreen(ST77XX_BLACK);
  tft.setTextColor(ST77XX_WHITE);
  tft.setTextSize(2);
  tft.setCursor(50, 120);
  tft.print(\"Hello, World!\");
  delay(2000);
}

Step 6

Don’t forget to plug in to your ESP32-C3 chip with a Type-C data cable.

Step 7

When ready to upload, click PlatformIO: Upload. At the bottom left.
